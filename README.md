# AiNurseAssist
IA de suporte à decisão clínica para médicos, com sugestões de exames e tratamentos baseados no diagnóstico.

AiNurseAssist (PI2025)

O AiNurseAssist é uma plataforma de Inteligência Artificial para suporte à decisão clínica, desenvolvida como parte do Projeto Integrador (PI2025).

O objetivo principal do sistema é auxiliar a equipe médica no processo de diagnóstico e tratamento. Com base no diagnóstico e no estado clínico atual do paciente, a ferramenta analisa os dados e fornece sugestões de:

Exames: Recomenda testes laboratoriais ou de imagem pertinentes para confirmar ou refinar o diagnóstico.

Medicamentos: Propõe opções de remédios e tratamentos alinhados com o quadro do paciente e as melhores práticas.

Este projeto visa otimizar o plano de tratamento, agilizar a tomada de decisão e servir como uma ferramenta de apoio para os profissionais de saúde.
